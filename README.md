# Rock Paper Scissors 🎮

A simple Rock-Paper-Scissors game written in C++.

## 💻 Features
- User vs Computer gameplay
- Input validation
- Random computer choices
- Score tracking

## 🛠️ Requirements
- A C++ compiler (e.g. g++, clang++)

## 🚀 How to Run

```bash
g++ rock-paper-scissors.cpp -o game
./game
